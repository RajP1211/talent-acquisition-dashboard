# Multi-step Job Application (React + Redux)

Quick multi-step job application form built with React and Redux Toolkit.

Features
- 6 separate step components
- Validation per-step
- Add/remove rows for Work Experience
- File upload stores filename
- Uses Bootstrap for styling

Run

1. Install deps

```bash
npm install
```

2. Start dev server

```bash
npm start
```

Notes
- Final submit logs payload to the browser console and shows an alert.
- This scaffold uses Redux Toolkit in `src/features/formSlice.js`.
